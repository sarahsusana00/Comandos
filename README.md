# Comandos
# Glosario de comandos


~~~bash
help: despliega una lista de comandos incluidos en el shell, si se agrega el nombre de una funcion te dara mas informacion sobre esa funcion en especifico

cd "nombre_directorio"/"..": avanza al directorio que se especifique, o en su defecto para retroceder al directorio anterior cuando se ponen los dos puntos

ls: nos muestra los archivos en el directorio actual

pwd: imprime el directorio en el que se esta trabajando

mkdir: crea un nuevo directorio

echo: lo usamos para escribir texto en un archivo

rmdir: borrar directorio


rm: elimina un archivo, usamos -rf para forzar que se elimine
./-ruta relativa

.-retrocedes en el album de directorios

echo-imprimir
 hacer un programa en batch /programar en terminal

history: despliega el historial de comandos usados en la terminal

git --version/-v: muestra la version de git instalada xd

git init: inicia un repositorio nuevo de git en el directorio actual

explorer: invoca el explorador de Windows, si se agrega un espacio seguido de un punto abre el directorio actual

. code: invoca Visual Studio Code, si se agrega un espacio seguido de un punto abre el directorio actual

git status: muestra los archivos en el staging area y los archivos que han sufrido cambios y que no se estan trackeando. Tambien brinda opciones generales sobre que se puede hacer con ellos

git add: agrega el archivo especificado al staaging ground para hacerlo commit 

git diff: muestra cambios de un archivo en diferentes commits

git restore: restaura un archivo a una version anterior

git commit: guarda una nueva version de los archivos en el staging ground

git config --global user.email/user.name: establece el correo y nombre de usuario de quien estara trabajando en el repositorio

git log --oneline: muestra una lista de los commits que se han realizado en el repositorio

clear: limpia la consola 

git branch "nombre": crea una nueva rama con el nombre especificado

git checkout "nombre": cambia de rama a la rama especificada 

git merge -nombredelacarpeta- crea ramas
~~~
